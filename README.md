> retains its value until used in "(let (x 1)(+ x 1))"-like format

fred

(- a 3)
> returns -2

(* a 3)
> returns 3

(/ a 3)
> returns 0.333333

(/ a 0)
> returns error message

(if #f 10 13)
> returns 13

(if #t 21 13)
> returns 21

###extra notes:

> variable clearing done as specified
